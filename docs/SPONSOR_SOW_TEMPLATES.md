## 3) Sponsored event formats (approved catalog)

Format A: Secure Coding Sprint
- Duration: 45 min
- Team size: 1-3
- Deliverable: patched code + brief explanation
- Winner selection: score rubric (correctness, security, clarity)

Format B: API Security Bug Hunt (sandbox)
- Duration: 60 min
- Scope: sponsor-provided intentionally vulnerable API instance
- Deliverable: report with repro + impact + fix

Format C: Cloud Security Quest
- Duration: 30-45 min
- Scope: IAM misconfig puzzle, logging triage, key rotation drill
- Deliverable: checklist completion + evidence capture

Format D: Reverse Engineering Micro-Challenge
- Duration: 30 min
- Scope: binary/string/static analysis puzzle
- Deliverable: flag + reasoning

Format E: Career Accelerator Booth
- Duration: continuous
- Scope: resume triage, portfolio check, mock interview bursts
- Deliverable: shortlist tokens or interview invites

Non-approved format
- Pure sales presentation without participant interaction
