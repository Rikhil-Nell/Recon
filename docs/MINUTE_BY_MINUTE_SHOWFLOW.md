# Minute-by-Minute Showflow - VIT-AP CyberSec Fest 2026

This is the control script for stage, announcements, and timing discipline.

## Global rules

- Timekeeping is strict; all hosts follow countdown cards: T-10, T-5, T-2, CUT.
- Any session slip > 7 min triggers schedule recovery plan.
- Every major transition must be reflected in app + campus screens within 3 minutes.

## Announcement templates (ready scripts)

Opening safety line
- "All security challenges are sandboxed and in-scope only. Any activity outside listed targets is prohibited and disqualifiable."

Queue balancing line
- "If your current queue exceeds 25 minutes, check the app for lower-wait zones now open."

Overnight wellness line
- "Hydrate, take short breaks, and use the rest area if needed. Sustained performance wins overnight rounds."

Delay line
- "We are in a technical reset for this zone. Next update in 10 minutes via app and display screens."

## Recovery playbook (time slip)

- Slip <= 7 min: compress transition buffers
- Slip 8-20 min: cut non-essential stage filler
- Slip > 20 min: invoke session swap with backup block

## Stage roles per session

- Emcee
- Timekeeper
- AV operator
- Speaker handler
- Floor manager

No session runs unless all five roles are present.
